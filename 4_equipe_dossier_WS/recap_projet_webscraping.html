01

Spider Jumia SN — Fonctionnel ✅

Méthode d'inspection utilisée : Scrapy Shell pour identifier les vrais sélecteurs CSS sur le site réel.

🔍 Découverte clé Jumia expose ses avis via une URL API interne :
/catalog/productratingsreviews/sku/{SKU}/
Le SKU est extrait depuis l'attribut data-sku du formulaire d'achat.
Python — Sélecteurs CSS validés
# Nom du produit
response.css("h1.-fs20.-pts.-pbxs::text").get()

# SKU du produit (clé pour accéder aux avis)
response.css("form#add-to-cart::attr(data-sku)").get()

# Conteneur d'un avis
review = response.css("article.-pvs.-hr._bet")

# Note de l'avis
review.css(".stars._m._al.-mvs::text").get()  # "4 out of 5"

# Texte du commentaire
review.css("p.-pvs::text").get()

# Date
review.css("span.-prs::text").get()  # "16-02-2026"

# Auteur
review.css("span:not(.-prs)::text").get()  # "par Ibrahima"
Résultats du premier scraping réussi
10 avis collectés sur 3 produits
Encodage UTF-8 corrigé (latin1 → utf-8)

Spider Google Maps — En attente d'activation API 🔑

Le spider est codé et fonctionnel. Il attend l'activation de la facturation Google Cloud pour utiliser l'API Places.

⚠️ Prérequis Activer la facturation sur console.cloud.google.com (gratuit jusqu'à 200$/mois). Ajouter la clé dans le fichier .env. Ne jamais partager la clé API publiquement.
API Google Places — Flux
# Étape 1 — Trouver le lieu par nom
GET /place/findplacefromtext/json
  ?input="Restaurant Le Lagon Dakar"
  &fields=place_id,name,rating

# Étape 2 — Récupérer les avis via place_id
GET /place/details/json
  ?place_id="ChIJxxxxxxxx"
  &fields=name,reviews,rating
  &language=fr
  # → Retourne max 5 avis (limite API gratuite)
