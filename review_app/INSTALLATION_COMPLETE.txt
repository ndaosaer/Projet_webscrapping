## 🔧 ÉTAPE 2 — BACKEND (API)

### 2.1 — Ajoute l'endpoint suggestions

Ouvre `api/main.py` :
```powershell
notepad ..\review_analyzer\api\main.py
```

**→ VA À LA FIN DU FICHIER** (avant le dernier commentaire) et **COLLE CE CODE** :

```python
# ══════════════════════════════════════════════════════════════════════════════
# ENDPOINT 6 — Suggestions pour autocomplétion
# ══════════════════════════════════════════════════════════════════════════════
@app.get(
    "/suggestions",
    tags=["Recherche"],
    summary="Suggestions de recherche pour autocomplétion",
)
def get_suggestions(
    query: str = Query(..., min_length=2, description="Texte de recherche (min 2 caractères)"),
    limit: int = Query(10, ge=1, le=20, description="Nombre de suggestions"),
    db: Session = Depends(get_session),
):
    """
    Retourne les noms de produits/lieux qui matchent la recherche.
    Utilisé pour l'autocomplétion dans la barre de recherche.
    """
    pattern = f"%{query}%"
    
    # Recherche dans product_name avec DISTINCT pour éviter les doublons
    results = (
        db.query(Review.product_name)
        .filter(Review.product_name.ilike(pattern))
        .distinct()
        .limit(limit)
        .all()
    )
    
    suggestions = [r[0] for r in results]
    
    return {
        "query": query,
        "count": len(suggestions),
        "suggestions": suggestions,
    }
```

Sauvegarde et **RELANCE L'API** :

```powershell
cd ..\review_analyzer
uvicorn api.main:app --reload --port 8000
```

Vérifie que l'API fonctionne : http://localhost:8000/docs

## 🐛 TROUBLESHOOTING

### Erreur "google_fonts not found"
```powershell
flutter clean
flutter pub get
flutter run -d chrome
```

### Erreur "getSuggestions not found"
→ Vérifie que l'endpoint est bien ajouté dans api/main.py
→ Relance l'API

### Page blanche
→ Ouvre la console Chrome (F12) et regarde les erreurs
→ Vérifie que l'API tourne sur http://localhost:8000

### Suggestions ne marchent pas
→ Teste l'endpoint : http://localhost:8000/suggestions?query=bou
→ Si ça retourne du JSON → c'est bon côté API
→ Si erreur → vérifie que tu as bien ajouté l'endpoint
